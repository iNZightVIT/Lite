Create variables
New columns or variables can be generated with this calculator like module. A set of controls can be used to generate a function for a new column. - A dropdown menu can be used to select variables from the current data set. - Several operators can be used to construct a function. - Twelve buttons can be used to add numbers to the function or delete part of the function. - In a text filed the name of the new variable can be specified. The default name is "new.name". A number is added to the name in case more than one variable is generated. - Next to the text field, the formula is displayed. - The "Create Variable" button adds the new variable to the data. - A information message tells whether the function can produce a variable.
