Export data
The current work data set can be downloaded. Select the file type in the dropdown menu at the to left and press the "DOWNLOAD" button. Several file types are available the data can be saved in.
  • txt
    A tab delimited text file.
  • csv
    A comma seperated text file.
  • RData
    A RData binary file.
  • RDS
    A RDS binary file.
Note, "RData" and "RDS" file are R only. They can be loaded into R with the load or readRDS function. They have the advantage that no information is lost. The txt files (txt and csv) are not able to store some information.
