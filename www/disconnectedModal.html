You have been disconnected from iNZight Lite!

This may have happened because you have been inactive for 15 minbutes, or there has been a problem with the application.

Below are some troubleshooting steps that may solve the problem. If you are still having trouble, please use the contact form to get in touch.

Troubleshooting

  • If the disconnect occurred after you uploaded some data …

    There may be an issue with the dataset, or Lite is running into trouble for some other reason.

    • If you haven't already, try loading the data again
    • If you are on a Mac, ensure you are not trying to upload a Numbers (.numbers) file, as Lite cannot read these
    • Try a different dataset. If you get the same issue, it might be an issue with Lite. In that case, please contact us.
  • If the disconnect occurred after you performed some action …

    This is likely an issue with Lite. Please contact us including the dataset and steps taken to reproduce the error.

Contact Support

If the above does not solve your problem, please contact us using the form: https://inzight.nz/support/contact/
